# Analysis of Algorithms

## Why Learn DSA?

Data Structures organize data efficiently.

Algorithms solve problems efficiently.

Together they improve application performance.

## Time Complexity

Measures execution time growth with input size.

### Common Complexities

O(1)  Constant Time

O(log n) Logarithmic Time

O(n) Linear Time

O(n log n) Efficient Sorting

O(n²) Nested Loop Algorithms

## Space Complexity

Measures memory usage.

## Asymptotic Notations

### Big O

Upper bound of growth.

### Omega

Lower bound of growth.

### Theta

Exact growth rate.

## Analysis Types

### Best Case

Minimum execution time.

### Average Case

Expected execution time.

### Worst Case

Maximum execution time.
