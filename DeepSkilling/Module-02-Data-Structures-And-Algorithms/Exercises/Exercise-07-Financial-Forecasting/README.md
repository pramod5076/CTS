# Exercise 7 - Financial Forecasting

## Objective

Develop a simple financial forecasting application to predict future investment value based on annual growth.

## Problem Statement

Given an initial investment amount, annual growth rate, and number of years, forecast the future value of the investment.

## Concepts Used

- Recursion
- Mathematical Computation
- Algorithm Analysis

## Learning Outcome

After completing this exercise, I understood how recursive algorithms can be applied to financial forecasting and prediction problems.
