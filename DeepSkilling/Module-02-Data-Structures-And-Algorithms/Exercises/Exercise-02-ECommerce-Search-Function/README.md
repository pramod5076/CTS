# Exercise 2 - E-Commerce Platform Search Function

## Objective

Develop a search function for an e-commerce platform using searching algorithms.

## Problem Statement

An e-commerce website contains a list of products. Users should be able to search for a product by name.

## Concepts Used

- Arrays
- Linear Search
- Time Complexity Analysis

## Learning Outcome

After completing this exercise, I understood how searching algorithms are used in real-world applications such as online shopping platforms.
