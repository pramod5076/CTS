# Linked List

## Introduction

A Linked List is a linear data structure where elements are connected using pointers.

Each node contains:

- Data
- Reference to next node

## Types

### Singly Linked List

Each node points to next node.

### Circular Singly Linked List

Last node points to first node.

### Doubly Linked List

Node contains previous and next references.

### Circular Doubly Linked List

First and last nodes are connected.

## Operations

- Insert
- Delete
- Search
- Traverse

# Array Time Complexity

| Operation | Complexity |
|-----------|------------|

# Searching Algorithms

## Linear Search

Best Case : O(1)

Average Case : O(n)

Worst Case : O(n)

## Binary Search

Best Case : O(1)

Average Case : O(log n)

Worst Case : O(log n)
