# Arrays

## Introduction

An Array is a collection of elements stored in contiguous memory locations.

Each element can be accessed using its index.

## Features

- Fixed size
- Fast access using index
- Contiguous memory allocation

## Array Traversal

Traversal means visiting every element of the array.

Example:

10 20 30 40 50

## Searching in Arrays

- Linear Search
- Binary Search

## When to Use Arrays

- Fixed-size data
- Fast random access
- Simple data storage
