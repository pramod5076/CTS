# Angular Component Commands

## Generate Component

```bash
ng generate component home
```

or

```bash
ng g c home
```

---

## Generate Component inside Folder

```bash
ng g c employee/details
```

---

## Generate Inline Template

```bash
ng g c home --inline-template
```

---

## Generate Inline Style

```bash
ng g c home --inline-style
```

---

## Skip Test File

```bash
ng g c home --skip-tests
```

---

## Build Project

```bash
ng build
```

---

## Run Project

```bash
ng serve
```
