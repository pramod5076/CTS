# Angular HTTP Client and APIs

## What is HttpClient?

HttpClient is Angular's built-in service used to communicate with backend REST APIs. It allows applications to send HTTP requests and receive responses asynchronously.

---

## Why Use HttpClient?

- Consume REST APIs
- Send and receive JSON data
- Supports Observables
- Built-in error handling
- Supports HTTP Interceptors
- Easy integration with backend services

---

## Common HTTP Methods

- GET – Retrieve data
- POST – Create new data
- PUT – Update existing data
- DELETE – Remove data

---

## Key Features

- HttpClientModule
- HttpClient Service
- Observables
- Promises
- Error Handling
- Retry Strategies
- HTTP Interceptors

---

## Advantages

- Simple API communication
- Asynchronous operations
- Reusable services
- Centralized request handling
- Improved security with interceptors

---

## Summary

Angular HttpClient simplifies communication with REST APIs while providing support for Observables, error handling, retry logic, and HTTP interceptors.
