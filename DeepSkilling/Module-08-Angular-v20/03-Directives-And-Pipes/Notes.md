# Directives and Pipes in Angular

## What are Directives?

Directives are special instructions in Angular that extend HTML by adding behavior to elements.

### Types of Directives

- Component Directives
- Structural Directives
- Attribute Directives
- Custom Directives

---

## Structural Directives

Structural directives modify the DOM layout.

Examples

- *ngIf
- *ngFor
- *ngSwitch

---

## Attribute Directives

Attribute directives change the appearance or behavior of an existing element.

Examples

- ngClass
- ngStyle
- ngModel

---

## Pipes

Pipes transform data before displaying it in the template.

Examples

- date
- uppercase
- lowercase
- currency
- number
- percent

---

## Custom Directives

Angular allows developers to create reusable custom directives.

---

## Custom Pipes

Custom pipes help transform application-specific data.

---

## Advantages

- Improves code readability
- Promotes code reuse
- Simplifies templates
- Easy data formatting

---

## Summary

Directives control DOM behavior, while Pipes transform data for display.
