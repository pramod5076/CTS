# Angular Routing and Navigation

## What is Routing?

Routing enables navigation between different views or components in a Single Page Application (SPA) without reloading the entire page.

---

## Why Routing?

- Navigation between pages
- Better User Experience
- URL-based Navigation
- Supports Lazy Loading
- Route Protection

---

- RouterModule
