# Introduction to Angular

## What is Angular?

Angular is an open-source front-end framework developed and maintained by Google for building dynamic Single Page Applications (SPAs).

---

## Features of Angular

- Component-Based Architecture
- TypeScript Support
- Dependency Injection
- Two-Way Data Binding
- Routing
- Lazy Loading
- Reactive Forms
- Built-in HTTP Client
- RxJS Support

---

## Advantages

- High Performance
- Scalable
- Reusable Components
- Large Community
- Enterprise Ready
