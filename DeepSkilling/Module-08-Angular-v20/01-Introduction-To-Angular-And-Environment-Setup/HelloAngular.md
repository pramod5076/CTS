# Angular CLI Commands

## Install CLI

```bash
npm install -g @angular/cli
```

---

## Version

```bash
ng version
```

---

## Create Project

```bash
ng new my-app
```

---

## Run

```bash
ng serve
```

---

## Production Build

```bash
ng build
```

---

## Generate Component

```bash
ng generate component home
```

---

## Generate Service

```bash
ng generate service employee
```

---

## Generate Module

```bash
ng generate module admin
```

---

## Run Tests

```bash
ng test
```

---

## Build for Production

```bash
ng build --configuration production
```
