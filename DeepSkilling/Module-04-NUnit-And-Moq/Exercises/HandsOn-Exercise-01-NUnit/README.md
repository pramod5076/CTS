## Objective

Learn the fundamentals of automated testing using NUnit framework.

## Concepts Covered

* Automated Testing
* Unit Testing
* Test Fixture
* Test Case
* Assertions
* Test Driven Development (TDD)

## Tools Used

* Visual Studio
* NUnit
* NUnit Test Adapter

## Learning Outcome

Successfully created and executed NUnit test cases to validate application functionality.
