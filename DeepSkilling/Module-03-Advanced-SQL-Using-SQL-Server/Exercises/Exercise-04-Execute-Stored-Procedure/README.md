# Exercise 4 - Execute Stored Procedure

## Objective

Execute an existing stored procedure and verify that data is inserted successfully.

## Problem Statement

A stored procedure named sp_InsertEmployee has already been created. Execute the procedure by passing employee details as parameters.

## Concepts Used

* EXEC Statement
* Stored Procedures
* Parameter Passing

## Learning Outcome

Learned how to execute stored procedures and insert records using parameters.
