# Module 6 - ASP.NET Core 8.0 Web API

## Overview

ASP.NET Core Web API is Microsoft's framework for building RESTful HTTP services. It enables developers to create scalable, secure, and high-performance backend APIs that can be consumed by web, mobile, desktop, and cloud applications.

---

## Topics Covered

- Introduction to Web APIs
- REST vs SOAP
- Setting up .NET 8 Development Environment
- ASP.NET Core Web API Project Structure

---

## Learning Objectives

After completing this module, you will be able to:

- Understand Web API fundamentals.
- Differentiate REST and SOAP services.
- Configure the .NET 8 development environment.
- Understand the ASP.NET Core Web API project structure.
- Build modern backend APIs using ASP.NET Core.
