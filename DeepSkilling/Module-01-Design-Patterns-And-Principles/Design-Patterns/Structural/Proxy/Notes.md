# Proxy Pattern

## Definition

The Proxy Pattern provides a placeholder or representative object that controls access to another object.

The proxy performs additional operations before delegating requests.

## Why Use Proxy?

Useful for:

* Security
* Access Control
* Logging
* Lazy Loading
* Performance Optimization

## Advantages

* Controlled access
* Improved security
* Better resource management

## Real World Example

An ATM machine acts as a proxy between a customer and a bank account.
